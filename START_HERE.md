## ğŸ†˜ Troubleshooting

### Can't See Anything?

- Check all files are in the right folders
- Check browser console (F12) for errors
- Verify JavaScript is enabled in browser

### Cards Not Flipping?

- Check `js/surgery.js` is in the right location
- Check browser console for errors
- Clear browser cache and refresh

### Changes Not Showing on GitHub Pages?

- Wait 2-3 minutes after committing
- Do a hard refresh: `Ctrl+Shift+R` (Windows) or `Cmd+Shift+R` (Mac)
- Check you're visiting the right URL (.github.io, not github.com)
